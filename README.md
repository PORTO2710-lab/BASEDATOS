# 🗄️ Base de Datos - Sistema CRUD Interactivo

Un sistema de gestión de usuarios robusto que permite realizar operaciones de creación, lectura, actualización y borrado (CRUD) en una base de datos relacional.

## 🚀 Demo en vivo
Accede al sistema aquí: [https://basedatos-pbl3.onrender.com/](https://basedatos-pbl3.onrender.com/)

## ✨ Funcionalidades
* **Listado en tiempo real:** Visualización instantánea de todos los usuarios registrados.
* **Gestión de Usuarios:** Formularios dedicados para agregar y editar nombres y correos.
* **Eliminación Segura:** Confirmación previa antes de borrar registros de la base de datos.
* **Interfaz Limpia:** Diseño Dark Mode optimizado para entornos administrativos.

## 🛠️ Stack Tecnológico
* **Lenguaje:** PHP 8.x
* **Base de Datos:** MySQL / MariaDB
* **Diseño:** CSS3 (Custom Styles)
* **Despliegue:** Docker + Render

## 📂 Estructura del Proyecto
* `index.php`: Panel principal y visualización de la tabla.
* `agregar.php`: Interfaz para nuevos registros.
* `editar.php`: Lógica de actualización de datos.
* `eliminar.php`: Script de borrado por ID.
* `conexion.php`: Configuración del enlace a la DB.
* `database.sql`: Estructura de la tabla de usuarios.
